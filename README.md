# Portfolio Hoàng Văn Vinh

Đây là trang web portfolio cá nhân của Hoàng Văn Vinh, một nhà thiết kế đồ họa kỹ thuật số.

## Mô tả

Trang web này được xây dựng bằng HTML và CSS thuần túy, với thiết kế hiện đại, sáng tạo và tối giản nhưng đầy màu sắc. Nó tuân thủ các thực hành tốt nhất về UX/UI, bao gồm:

- Phân cấp thị giác mạnh mẽ
- Khoảng cách sạch sẽ
- Typography hiện đại
- Thiết kế đáp ứng

## Cấu trúc

- `index.html`: Trang chính chứa nội dung portfolio
- `styles.css`: Tệp CSS cho kiểu dáng và bố cục

## Cách sử dụng

Mở tệp `index.html` trong trình duyệt web để xem portfolio.
